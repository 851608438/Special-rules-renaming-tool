# Special-rules-renaming-tool
特殊规则的重命名工具

# 使用方式
1. 将py文件复制到你要处理的文件夹内
2. py XXX.py


各python文件功能如下：

# change files name
搜索当前文件夹及子文件夹所有文件的文件名（包括文件夹的名称），从右向左匹配第一个"_0"或"_1"或"_2"或"_3"或"_4"或"_5"或"_6"或"_7"或"_8"或"_9"，并将其修改成"_00"或"_01"或"_02"或"_03"或"_04"或"_05"或"_06"或"_07"或"_08"或"_09"。并且匹配时只匹配从右向左4个字符（不包括扩展名）

# file length check
检查指定的源目录中的所有文件，找出那些文件名长度（以字节计）超过255字节的文件

# change files name1
搜索当前文件夹及子文件夹所有文件的文件名（包括文件夹的名称），从右向左匹配第一个"(0)"或"(1)"或"(2)"或"(3)"或"(4)"或"(5)"或"(6)"或"(7)"或"(8)"或"(9)"，并将其修改成"(00)"或"(01)"或"(02)"或"(03)"或"(04)"或"(05)"或"(06)"或"(07)"或"(08)"或"(09)"。并且匹配时只匹配从右向左4个字符（不包括扩展名）

# rename
将当前文件夹中的子文件夹中的所有文件，按名称递增的顺序，依次重命名为“子文件夹名_001”、“子文件夹名_002”、“子文件夹名_003”...以此类推。且不能改变文件的扩展名。

# move
将当前文件夹中的子文件夹中的所有文件，全部移动到当前文件夹，移动完毕后将子文件夹删除。如果移动过程中遇到有文件的文件名重复，则停止移动该文件且保留该子文件夹。

# rename1
将当前文件夹中的子文件夹中的所有文件，按名称递增的顺序，依次重命名为“子文件夹名_001”、“子文件夹名_002”、“子文件夹名_003”...以此类推。不能改变文件的扩展名。且遇到多层子文件夹应按下述格式进行命名：“子文件夹名_第二层子文件夹名_第三层子文件名_..._001”以此类推

# file length check1
检查当前文件夹及子文件夹中的所有文件的文件名，发现有文件名长度超过255个字符的文件，将该文件名和路径写入到“file_length_check.txt”文件中。并将该文件复制到指定的文件夹。

# file length check2
检查当前文件夹及子文件夹中的所有文件的文件名，发现有文件名长度超过255个字节的文件，将该文件的文件名的中间部分替换成"..."。且中间部分的字节数必须为总文件名字节数的一半。
